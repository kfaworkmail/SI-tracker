# PRD — СИ-Трекер

## 1. Overview

**СИ-Трекер** — веб-приложение для учета средств измерений (СИ) на предприятии. Система предназначена для метрологов, материально ответственных лиц и руководителей подразделений, которым необходимо вести реестр приборов, контролировать сроки поверки и получать оперативную статистику.

Проект представляет собой учебный MVP, который можно реализовать за 1–2 дня с помощью AI-инструментов разработки (v0.dev, Cursor, Bolt.new).

---

## 2. Problem Statement

На многих предприятиях учет средств измерений ведется в Excel-файлах. Такой подход создает ряд проблем:

* данные сложно искать и фильтровать;
* отсутствует централизованная карточка прибора;
* нет автоматического контроля сроков поверки;
* сложно получать сводную статистику;
* повышен риск ошибок при ручном редактировании;
* затруднена совместная работа нескольких пользователей.

СИ-Трекер переводит учет СИ из Excel в специализированное веб-приложение с удобным интерфейсом, структурированными данными и автоматическим контролем сроков поверки.

## 4. Target Users

### Метролог

Основной пользователь системы.

Обязанности:

* ведение реестра СИ;
* обновление данных;
* контроль сроков поверки;
* импорт и экспорт данных.

### Материально ответственное лицо (МОЛ)

Использует систему для просмотра закрепленных за ним приборов.

Обязанности:

* контроль местонахождения СИ;
* просмотр статуса поверки;
* контроль состояния оборудования.

### Руководитель подразделения

Использует систему только для просмотра информации и статистики.

Обязанности:

* контроль обеспеченности подразделения СИ;
* контроль просроченных поверок;
* получение сводной информации.

## 11. Out of Scope

В первую версию продукта не входят:

* полноценная LIMS-система;
* история всех поверок и калибровок;
* хранение файлов сертификатов;
* генерация сертификатов калибровки;
* электронная цифровая подпись (ЭЦП);
* интеграция с государственным реестром СИ;
* интеграция с ERP, SAP и 1С;
* работа со штрихкодами и QR-кодами;
* автоматические уведомления по email и SMS;
* мобильное приложение;
* расширенная ролевая модель доступа;
* интеграция с лабораторным оборудованием.
